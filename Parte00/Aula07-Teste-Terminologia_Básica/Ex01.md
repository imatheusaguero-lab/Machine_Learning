III. Utiliza conceitos da área da genética para gerar as respostas

## Pergunta 3:
Mineração de dados é o processo de criar algoritmos para extrair conhecimento de bases de dados.
**Resposta:**
Falso
**Porque:**
Mineração de dados é o processo de extrair conhecimento de bases de dados, e não de criar algoritmos.
