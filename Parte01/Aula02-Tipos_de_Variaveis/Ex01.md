# Variaveis
- Numéricas
- Categóricas
    : Representam categorias. Tipo STRING

## Numéricas
- Contínua
    Tipo FLOAT
    - Números reais
    - Temperatura, altura, peso, salario
- Discreta
    Tipo INTEIRO
    - Conjunto de valores finito (inteiros)
    - Contagem de alguma coisa.

## Categóricas
- Nominal
    - Dados não mensuráveis
    - Sem ordenação: cor dos olhos, gênero, id, nome.
- Ordinal
    - Categorização sob uma ordem
    - Tamanho: P, M, G e GG
